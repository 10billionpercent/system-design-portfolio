# System Design Portfolio

A collection of system design diagrams for projects I have built, am currently building, or have fully planned at the architecture level.

This repository is a high-level view of how I approach products beyond just code or UI. It focuses on structure, data flow, feature logic, and how different parts of a system connect together.

## What's Inside

Each project has its own folder.

For now, each folder includes - 

- System Diagram
- Sequence Diagram

As projects evolve, more diagrams may be added over time.
